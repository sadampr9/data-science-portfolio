# Portafolio de Ciencia de Datos – Sadam Pérez Romero

Científico de datos con formación de máster en Inteligencia Artificial, 
enfocado en análisis de datos, machine learning y NLP sobre datos públicos 
y contextos sensibles.

## Áreas de experiencia
- Clasificación y predicción
- Procesamiento de lenguaje natural (NLP)
- Análisis de sentimiento
- Visualización de datos
- SQL y análisis exploratorio

## Tecnologías
- Python (Pandas, NumPy, Scikit-learn)
- Visualización: Matplotlib, Seaborn
- SQL
- Notebooks (Jupyter)

## Proyectos destacados
- Análisis de sentimiento en comentarios periodísticos
- Clasificación supervisada sobre datos públicos
- Modelos predictivos
- Introducción a visión artificial (clasificación de imágenes)

Todos los proyectos utilizan datos públicos y tienen fines académicos.
